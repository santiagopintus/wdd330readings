Window Object

Every JavaScript environment has a global object. Any variables or functions created in the global scope are properties or methods of this object, respectively. In a browser environment, the global object is the window object. Thus, the window object only exists in a browser environment.

For instance:

//Declaring a global variable
student = "Santiago";
//Then
console.log(window.student) // outputs: "Santi"
//Now the global variable student is a property of the window object.

An interesting property I learned of is the navigator property that returns a reference of the Navigator object. By accessing the userAgent property of the Navigator object, we can get information about the user browser or operative system.

There are many other useful methods and properties of the window object that will help me to accomplish my tasks.
