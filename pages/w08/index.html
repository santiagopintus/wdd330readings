CSS Transforms

The transform CSS property allows to modify an element in the following ways:

  • Rotate
  • Scale
  • Skew
  • Translate

To rotate I must indicate the degrees to rotate the element as a function parameter: transform: rotate(45deg);

To scale, I must indicate a value. A percent less than 100% or a single number less than 1 will cause the element to shrink. If I want to scale an element in the X and Y axes separately, I must pass two values: transform: scale(20%, 120%);

Skew: It skews an element on the 2D plane. The first value will skew on the x-axis and the second on the y-axis: transform: skew(15deg, 30deg);

Translate: Moves an element on the 2D plane. The first value moves the element on the x-axis, and the second on the y-axis. If only one value is passed, then it moves on both axes: transform: translate(10px, 2px);

CSS Transitions

Transitions provide a way to control the speed of CSS animations when properties are changed. By default changes occur immediately, transitions make this change smooth for a nicer effect.

Code example of transform + transition

.square {
  transition: background-color 200ms ease;
}
.square:hover {
  background-color: #000000;
}

Here, I added a transition to the background-color on the hover event.
