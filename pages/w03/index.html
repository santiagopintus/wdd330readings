"This" Keyword

In OOP or "Object Oriented Programming" objects are used to represent entities of the real world. As real entities, objects have properties and in many cases, the capacity to act. A Javascript object performs an action through function properties called methods. To access the data inside of an object, its method should use the keyword this to refer to the object and then access one of its properties. For instance: 

let student = {
  name: "Santiago",
  lastName: "Pintus",
  currentCourse: "WDD330",
  study() {
    console.log(`Currently studying ${this.currentCourse}`)
  }
}
