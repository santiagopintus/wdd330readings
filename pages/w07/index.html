AJAX

Asynchronous JavaScript and XML: This is the standard for fetching data from a server and reloading only part of the page to update the shown data. The fetch API is a way to execute these requests. This is the basic structure: 

fetch("https://urlexample.com")
  .then(//Do something after receiving a response)
  .catch(//Do something if there's an error on the fetch)
  .finally(//Finally execute other code after the fetch ends)

There are different methods to perform various actions on an HTTP request. The methods are:

  • GET: For retrieving data from the server.
  • POST: For sending data from the frontend to the server
  • PUT: Generally for updating data in the server
  • PATCH: For updating partially a resource
  • DELETE: For deleting a resource
