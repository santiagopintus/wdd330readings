Javascript Lesson

Object-oriented programming is a way to create different objects from one model object. The objects created using this model, are called instances of the object. This model object is like a blueprint. Constructor functions are a way to define this blueprint. Using classes is a way to create constructor functions?? What is the advantage of using classes instead of just constructor functions? "This" is a JavaScript keyword that allows me to access the object, class, or constructor from a method of the object. Modularizing JavaScripts apps is a great way to organize better the code. This way, I can make each module have a specific responsibility and also provides mantainability to the app in the future.
