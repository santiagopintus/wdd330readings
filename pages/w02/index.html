Functions

Functions are a super important part of JavaScript. There are several ways to declare functions and several ways to call them. Functions can be declared using function declaration method as a named function: 

function sum(num1, num2){}
Anonymous function (assigned to a variable): 
const sum = function(num1, num2){}
Also as an arrow function: 
const sum = (num1, num2) => {}

Hoisting

Hoisting is a little tricky concept but as I understand Hoisting is the fact that JavaScript moves all variables and function declarations to the top of the current scope. The function can be called before its declaration, if declared as a function declaration. The declaration of variables declared with "var" keyword is moved to the top as well but it is not hoisted its value assignment, so it can be only used after it was assigned a value.
